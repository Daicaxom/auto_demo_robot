Hướng Dẫn Cài Đặt Robot Framework và Thư Viện

Hướng dẫn này sẽ giúp bạn cài đặt Robot Framework và các thư viện cần thiết để bắt đầu viết các test case tự động hóa.

1. Điều Kiện Tiên Quyết

Trước khi cài đặt Robot Framework, bạn cần đảm bảo rằng hệ thống của bạn đã cài đặt Python và pip.

1.1. Kiểm tra Python

Mở terminal (hoặc Command Prompt trên Windows) và chạy lệnh sau để kiểm tra xem Python đã được cài đặt chưa:

python --version
# hoặc
python3 --version

Nếu Python đã được cài đặt, bạn sẽ thấy phiên bản Python được hiển thị. Nếu chưa, bạn cần tải và cài đặt Python từ trang web chính thức của Python.

1.2. Kiểm tra pip

Pip là trình quản lý gói cho Python, được sử dụng để cài đặt và quản lý các thư viện Python. Kiểm tra pip bằng lệnh:

pip --version
# hoặc
pip3 --version

Nếu pip chưa được cài đặt, bạn có thể cài đặt nó bằng cách chạy lệnh sau trong terminal:

# Đảm bảo bạn đã cài đặt Python trước khi chạy lệnh này
python -m ensurepip --default-pip
# hoặc (nếu python là python3)
python3 -m ensurepip --default-pip

2. Cài Đặt Robot Framework

Sau khi đã cài đặt Python và pip, bạn có thể cài đặt Robot Framework bằng lệnh pip:

pip install robotframework
# hoặc
pip3 install robotframework

Sau khi cài đặt thành công, bạn có thể kiểm tra phiên bản Robot Framework bằng lệnh:

robot --version

3. Cài Đặt Các Thư Viện Robot Framework Quan Trọng

Để sử dụng Robot Framework cho các mục đích khác nhau như kiểm thử web, API, và database, bạn cần cài đặt thêm các thư viện tương ứng. Dưới đây là một số thư viện phổ biến và cách cài đặt chúng:

3.1. SeleniumLibrary (Kiểm thử Web)

SeleniumLibrary là thư viện mạnh mẽ để tự động hóa trình duyệt web.

pip install robotframework-seleniumlibrary
# hoặc
pip3 install robotframework-seleniumlibrary

3.2. RequestsLibrary (Kiểm thử API)

RequestsLibrary giúp bạn thực hiện các kiểm thử API dễ dàng.

pip install robotframework-requests
# hoặc
pip3 install robotframework-requests

3.3. DatabaseLibrary (Kiểm thử Database)

DatabaseLibrary cho phép bạn tương tác và kiểm thử cơ sở dữ liệu.

pip install robotframework-databaselibrary
# hoặc
pip3 install robotframework-databaselibrary

3.4. Các thư viện khác (tùy chọn)

Bạn có thể cài đặt thêm các thư viện khác tùy theo nhu cầu dự án của bạn. Ví dụ:

  • Faker: Sinh dữ liệu giả mạo cho test. pip install Faker
  • PyYAML: Xử lý file YAML. pip install PyYAML
  • webdrivermanager: Quản lý driver cho trình duyệt web. pip install webdrivermanager
  • robotframework-metrics: Tạo báo cáo metrics cho Robot Framework. pip install robotframework-metrics
  • pymysql hoặc mysql-connector-python: Kết nối đến MySQL database. pip install pymysql hoặc pip install mysql-connector-python

Bạn có thể cài đặt tất cả các thư viện cần thiết cùng một lúc bằng cách sử dụng file requirements.txt (nếu có):

pip install -r requirements.txt
# hoặc
pip3 install -r requirements.txt

4. Xác Minh Cài Đặt

Sau khi cài đặt Robot Framework và các thư viện, bạn nên xác minh rằng mọi thứ đã được cài đặt đúng cách. Bạn có thể tạo một file test đơn giản (ví dụ: verify_install.robot) với nội dung sau:

***Settings***
Library    SeleniumLibrary

***Test Cases***
Verify Robot Framework Installation
    Log To Console    Robot Framework đã được cài đặt thành công!
    Open Browser    https://www.google.com    chrome
    Close Browser

Sau đó, chạy test case này bằng lệnh:

robot verify_install.robot

Nếu test case chạy thành công, bạn sẽ thấy thông báo "Robot Framework đã được cài đặt thành công!" trong console và một trình duyệt Chrome sẽ mở ra và đóng lại. Điều này xác nhận rằng Robot Framework và SeleniumLibrary đã được cài đặt và hoạt động bình thường.

Chúc mừng bạn đã cài đặt thành công Robot Framework và các thư viện cần thiết! Bây giờ bạn có thể bắt đầu xây dựng các kịch bản kiểm thử tự động hóa của mình.
